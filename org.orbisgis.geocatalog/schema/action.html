org.orbisgis.geocatalog.Action

Example:

    < extension point="org.orbisgis.geocatalog.Action" >
        < menu id="org.orbisgis.geocatalog.File" text="File" icon="/org/orbisgis/geocatalog/home.png" />
        < menu id="org.orbisgis.geocatalog.Help" text="Help" icon="/org/orbisgis/geocatalog/help.png" />
        < action id="org.orbisgis.geocatalog.NewResource" text="New Resource" parent="org.orbisgis.geocatalog.File" class="org.orbisgis.geocatalog.NewResourceAction" />
        < action id="org.orbisgis.geocatalog.Exit" text="Exit" parent="org.orbisgis.geocatalog.File" class="org.orbisgis.geocatalog.ExitAction" />
        < action id="org.orbisgis.geocatalog.About" text="About OrbisGIS" parent="org.orbisgis.geocatalog.Help" class="org.orbisgis.geocatalog.AboutAction" >
          </ action>
      </ extension>

    Content:

    Element nameminOccursmaxOccurs
    menu0unbounded
    action1unbounded

    Attributes:

    Attribute nameRequiredDescription
    point required Id of the extension point
    id optional Id of the extension. For debuging purposes only
    name optional Name of the extension. For debuging purposes only

    menu

    Installs a parent menu in the geocatalog

    Attributes:

    Attribute nameRequiredDescription
    id required Id of the menu, referenced by other attributes
    text required Text to be shown in the menu
    parent optional References the id of another menu
    icon optional Path to the icon to show in the menu. It has to be in the classpath of the plug-in

    action

    Installs an action in the geocatalog

    Attributes:

    Attribute nameRequiredDescription
    class required Class that implements the org.orbisgis.geocatalog.IGeocatalogAction interface
    id required Id of the action. Must be unique
    text required Text that will appear in the menu that triggers this action
    parent optional Menu where this action will be installed
    icon optional Icon associated to the action
