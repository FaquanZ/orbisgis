org.orbisgis.geoview.toc.LayerAction

Description:

This extension point is used to add contextual actions in the layer tree. This actions will be shown when the user right clicks in a layer

Example:

    < extension point=" org.orbisgis.geoview.toc.LayerAction" id=" org.orbisgis.geoview.toc.LayerManagement" >
        < action id=" org.orbisgis.geoview.toc.RemoveLayerAction" text=" Remove layer" group=" org.orbisgis.geoview.toc.LayerManagement" class=" org.orbisgis.geoview.toc.RemoveLayerAction" />
        < action id=" org.orbisgis.geoview.toc.GroupLayersAction" text=" Group layers" group=" org.orbisgis.geoview.toc.LayerManagement" class=" org.orbisgis.geoview.toc.GroupLayersAction" />
        < action id=" org.orbisgis.geoview.toc.CreateGroupAction" text=" New group" group=" org.orbisgis.geoview.toc.LayerManagement" class=" org.orbisgis.geoview.toc.CreateGroupAction" />
    </ extension>

Attributes:

Attribute nameRequiredDescription
id required id of the action. For internal use. Must be unique
group required The group of the action. It will be used to group graphically the actions that have the same value in this attribute
text required Text to display to the user
icon optional Icon to display to the user
class required Class that implements the action. It has to implement org.orbisgis.geoview.toc.ITocAction
