org.orbisgis.geocatalog.DND

Description:

Extension point that lets the management of events of drag and drop. It can also manage events started in this control and ended in another

Example:

    < extension point=" org.orbisgis.geocatalog.DND" >
        < dnd class=" org.orbisgis.geocatalog.BasicDnD" />
    </ extension>

Content:

Element nameminOccursmaxOccurs
dnd1unbounded

Attributes:

Attribute nameRequiredDescription
point required Id of the extension point
id optional Id of the extension. For debuging purposes only
name optional Name of the extension. For debuging purposes only

dnd

Installs a drag and drop handler

Attributes:

Attribute nameRequiredDescription
class required Class that implements the org.orbisgis.resourceTree.IResourceDnD interface
source optional If the source is specified, this extension will manage drag and drop events draged in the source and droped in the extension point. If this attribute is not specified, the extension will manage all the drag and drop events originated and droped in the same component
