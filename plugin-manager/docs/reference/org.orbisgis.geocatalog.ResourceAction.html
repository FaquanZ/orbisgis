org.orbisgis.geocatalog.ResourceAction

Description:

Action to execute on resouces in the geocatalog

Example:

    < extension point=" org.orbisgis.geocatalog.ResourceAction" >
        < menu id=" org.orbisgis.geocatalog.file.New" text=" Add" />
        < action id=" org.orbisgis.geocatalog.NewResource" menuId=" org.orbisgis.geocatalog.file.New" menuGroup=" org.orbisgis.geocatalog.new" text=" Add..." icon=" /org/orbisgis/geocatalog/images/add.png" class=" org.orbisgis.geocatalog.actions.NewResourceAction" />
        < action id=" org.orbisgis.geocatalog.DeleteResource" menuGroup=" org.orbisgis.geocatalog.basic" text=" Delete resource" icon=" /org/orbisgis/geocatalog/images/remove.png" class=" org.orbisgis.geocatalog.actions.DeleteResource" />
        < action id=" org.orbisgis.geocatalog.ClearCatalog" menuGroup=" org.orbisgis.geocatalog.global" text=" Clear catalog" icon=" /org/orbisgis/geocatalog/images/bin_closed.png" class=" org.orbisgis.geocatalog.actions.ClearCatalogAction" />
        < action id=" org.orbisgis.geocatalog.ClearFolder" menuGroup=" org.orbisgis.geocatalog.basic" text=" Clear folder" icon=" /org/orbisgis/geocatalog/images/bin_closed.png" class=" org.orbisgis.geocatalog.actions.ClearFolderAction" />
    </ extension>

Attributes:

Attribute nameRequiredDescription
point required

menu

Installs a parent menu

Attributes:

Attribute nameRequiredDescription
id required Id of the menu, referenced by other attributes
text required Text to be shown in the menu
parent optional References the id of another menu
icon optional Path to the icon to show in the menu. It has to be in the classpath of the plug-in

action

Defines a contextual action of the geocatalog. The will be active when there is some resource selected.

Attributes:

Attribute nameRequiredDescription
id required id of the action. For internal use. Must be unique
menuGroup required The group of the action. It will be used to group graphically the actions that have the same value in this attribute
menuId optional Id of the menu where this action will be installed
text required Text to display to the user
icon optional Icon to display to the user
class required Class that implements the action. It has to implement org.orbisgis.geocatalog.IResourceAction
