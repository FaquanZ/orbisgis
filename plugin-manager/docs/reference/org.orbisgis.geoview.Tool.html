org.orbisgis.geoview.Tool

Description:

Tool to interactuate with the map

Example:

    < extension id=" org.orbisgis.geoview.NavigationTools" point=" org.orbisgis.geoview.Tool" >
        < menu id=" org.orbisgis.geoview.Tools" text=" Tools" />
        < toolbar id=" org.orbisgis.geoview.Navigation" text=" Navigation" />
        < tool id=" org.orbisgis.geoview.tools.ZoomIn" menuId=" org.orbisgis.geoview.Tools" text=" Zoom In" menuGroup=" org.orbisgis.geoview.tools.Selectable" icon=" /org/orbisgis/geoview/images/zoom_in.png" mouse-cursor=" /org/orbisgis/geoview/images/zoom_in.gif" toolbarId=" org.orbisgis.geoview.Navigation" selectable=" true" class=" org.orbisgis.tools.instances.ZoomInTool" />
        < tool id=" org.orbisgis.geoview.tools.ZoomOut" menuId=" org.orbisgis.geoview.Tools" text=" Zoom Out" menuGroup=" org.orbisgis.geoview.tools.Selectable" icon=" /org/orbisgis/geoview/images/zoom_out.png" mouse-cursor=" /org/orbisgis/geoview/images/zoom_out.gif" toolbarId=" org.orbisgis.geoview.Navigation" selectable=" true" class=" org.orbisgis.tools.instances.ZoomOutTool" />
        < tool id=" org.orbisgis.geoview.tools.Pan" menuId=" org.orbisgis.geoview.Tools" text=" Pan" menuGroup=" org.orbisgis.geoview.tools.Selectable" icon=" /org/orbisgis/geoview/images/pan.png" mouse-cursor=" /org/orbisgis/geoview/images/pan.gif" toolbarId=" org.orbisgis.geoview.Navigation" selectable=" true" class=" org.orbisgis.tools.instances.PanTool" />
    </ extension>

Content:

Element nameminOccursmaxOccurs
menu0unbounded
toolbar0unbounded
tool1unbounded

Attributes:

Attribute nameRequiredDescription
point required Id of the extension point
id optional Id of the extension. For debuging purposes only
name optional Name of the extension. For debuging purposes only

toolbar

Installs a toolbar

Attributes:

Attribute nameRequiredDescription
id required Id of the toolbar, referenced by other attributes
text required Text to be shown in the tool bar

menu

Installs a parent menu

Attributes:

Attribute nameRequiredDescription
id required Id of the menu, referenced by other attributes
text required Text to be shown in the menu
parent optional References the id of another menu
icon optional Path to the icon to show in the menu. It has to be in the classpath of the plug-in

tool

Installs the tool

Attributes:

Attribute nameRequiredDescription
class required Class that implements the tool interface: org.orbisgis.tools.Automaton
id required Id of the tool. Must be unique
text required Text that will appear in the menu that selects this tool
menuId optional Id of the menu where this tool will be installed
icon optional Icon that will appear in the menus or buttons that select this tool
mouse-cursor optional Mouse cursor to show when this tool is selected. If nothing is specified a default 'cross' cursor is shown
menuGroup optional The group to associate the tools in the menus. Different groups will be separated visually
toolbarId optional Id of the toolbar where this tool will be installed. It is also necessary to specify the icon attribute
selectable optional If the tool is selectable.
