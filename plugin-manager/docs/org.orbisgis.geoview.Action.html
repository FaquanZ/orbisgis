org.orbisgis.geoview.Action

Description:

Action that will be installed in the menu, tool bar, etc.

Example:

    < extension id=" org.orbisgis.geoview.NewLayer" point=" org.orbisgis.geoview.Action" >
        < menu id=" org.orbisgis.geoview.File" text=" File" />
        < action menuId=" org.orbisgis.geoview.File" id=" org.orbisgis.geoview.NewLayerAction" text=" Open" icon=" /org/orbisgis/geoview/add.png" class=" org.orbisgis.geoview.NewLayerAction" />
    </ extension>

Content:

Element nameminOccursmaxOccurs
menu0unbounded
toolbar0unbounded
action1unbounded

Attributes:

Attribute nameRequiredDescription
point required Id of the extension point
id optional Id of the extension. For debuging purposes only
name optional Name of the extension. For debuging purposes only

toolbar

Installs a toolbar

Attributes:

Attribute nameRequiredDescription
id required Id of the toolbar, referenced by other attributes
text required Text to be shown in the tool bar

menu

Installs a parent menu

Attributes:

Attribute nameRequiredDescription
id required Id of the menu, referenced by other attributes
text required Text to be shown in the menu
parent optional References the id of another menu
icon optional Path to the icon to show in the menu. It has to be in the classpath of the plug-in

action

Installs an action in the geoview

Attributes:

Attribute nameRequiredDescription
class required Class that implements the action interface interface. This can be org.orbisgis.geoview.IGeoviewAction if the action is in a geoview, org.orbisgis.geocatalog.IGeocatalogAction if the action is in the geocatalog
id required Id of the action. Must be unique
text required Text that will appear in the menu that triggers this action
menuId optional Id of the menu where this action will be installed
icon optional Icon associated to the action
toolbarId optional Id of the toolbar where this action will be installed. It is also necessary to specify the icon attribute
exclusiveGroup optional All the actions that have this attribute are toggle actions. This is just a user interface feature that will make the associated control to be "selected" instead of "clicked". All the actions that share the same value for this attribute will be mutually exclusive, this is, there can't be two of these actions selected at the same time
