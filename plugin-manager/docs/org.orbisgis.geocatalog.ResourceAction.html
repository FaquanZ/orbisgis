org.orbisgis.geocatalog.ResourceAction

Description:

Action to execute on resouces in the geocatalog

Example:

    < extension point=" org.orbisgis.geocatalog.ResourceAction" >
        < action id=" org.orbisgis.geocatalog.NewResource" group=" org.orbisgis.geocatalog.new" text=" New..." class=" org.orbisgis.geocatalog.resources.NewResourceAction" />
        < action id=" org.orbisgis.geocatalog.DeleteResource" group=" org.orbisgis.geocatalog.basic" text=" Delete resource" icon=" /org/orbisgis/geocatalog/remove.png" class=" org.orbisgis.geocatalog.resources.DeleteResource" />
        < action id=" org.orbisgis.geocatalog.ClearCatalog" group=" org.orbisgis.geocatalog.global" text=" Clear catalog" class=" org.orbisgis.geocatalog.resources.ClearCatalogAction" />
        < action id=" org.orbisgis.geocatalog.ClearFolder" group=" org.orbisgis.geocatalog.basic" text=" Clear folder" class=" org.orbisgis.geocatalog.resources.ClearFolderAction" />
    </ extension>

action

Defines a contextual action of the geocatalog. The will be active when there is some resource selected.

Attributes:

Attribute nameRequiredDescription
id required id of the action. For internal use. Must be unique
group required The group of the action. It will be used to group graphically the actions that have the same value in this attribute
text required Text to display to the user
icon optional Icon to display to the user
class required Class that implements the action. It has to implement org.orbisgis.geocatalog.IResourceAction
