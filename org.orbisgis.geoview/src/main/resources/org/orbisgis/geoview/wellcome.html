OrbisGIS is a Geographical Information System (GIS) dedicate to scientific spatial simulation. This cross-platform GIS is developped at french IRSTV institute (Research Institute on Urban Sciences and Technics, CNRS/FR-2488), in the context of the Urban Data Federative Research Project. This project has to do with Urban GIS: methods and tools for the spatial analysis of the urban environment. It is able to handle vectorial and raster spatial information.
