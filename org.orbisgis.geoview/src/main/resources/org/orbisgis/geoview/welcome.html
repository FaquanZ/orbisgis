OrbisGIS is a Geographical Information System (GIS) dedicate to scientific spatial simulation. This cross-platform GIS is developped at french IRSTV institute (CNRS/FR-2488). It is able to handle vectorial and raster spatial information.

OrbisGIS is free software distributed under version 3 of the GNU General Public License.
